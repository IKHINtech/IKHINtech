## 👨‍💻 About Me

I'm a Fullstack Software Engineer based in Indonesia with experience building web, mobile, and backend applications for real business operations.

Currently, I work on enterprise systems at **PT. Indo Retreading and Tire Service**, where I contribute to mobile applications, web applications, backend services, admin dashboards, reporting systems, and legacy system modernization.

- SaaS, multi-tenant systems, internal tools, and enterprise applications
- Linux servers, Docker, Nginx, observability, and deployment workflows
- Open-source tools and developer experience
